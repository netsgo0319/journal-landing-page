## 📋 빠른 참조

### 핵심 메시지
- **메인**: "감사를 강요하지 않는 유일한 일기 앱"
- **서브**: "힘든 하루를 견디는 당신에게 진정한 위로를 전달하는 AI 동반자"

### 주요 기능
1. 공감하는 AI 동반자
2. 부담 없는 입력
3. 생애주기별 맞춤 경험
4. 칭찬 시스템
5. 유연한 감사 찾기

### 기술 스택 (권장)
- **Frontend**: HTML/CSS/JS (정적) 또는 React + Vite
- **스타일링**: Tailwind CSS 또는 순수 CSS
- **배포**: GitHub Pages

### 예상 개발 기간
- **총 기간**: 2-3주
- **MVP**: 1-2주
- **최적화 및 배포**: 1주
