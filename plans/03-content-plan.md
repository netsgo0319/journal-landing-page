# 콘텐츠 계획

## 📝 콘텐츠 전략

### 톤 앤 매너 (Tone & Manner)
- **따뜻함**: 공감하고 이해하는 톤
- **진정성**: 오버하지 않고 진심 어림
- **명확성**: 복잡하지 않고 이해하기 쉬움
- **포용성**: 모든 사람을 환영하는 느낌

### 목표
- 프로젝트의 핵심 가치를 명확히 전달
- 사용자의 문제를 이해하고 공감
- 차별점을 강조하되 경쟁사를 비판하지 않음
- 기술적 세부사항보다는 사용자 가치 중심

## 📄 섹션별 콘텐츠

### 1. Hero 섹션

#### 헤드라인
**메인 헤드라인 (H1)**
```
감사를 강요하지 않는 유일한 일기 앱
```

**서브 헤드라인**
```
힘든 하루를 견디는 당신에게 진정한 위로를 전달하는 AI 동반자
```

**설명 텍스트**
```
기존 일기 앱들은 감사를 찾으라고 요구합니다. 
하지만 우리는 다릅니다. 
당신이 여기 온 것만으로도 충분하다고 말합니다.
```

#### CTA 버튼
- **Primary**: "GitHub에서 보기" → GitHub 저장소 링크
- **Secondary**: "데모 보기" → 데모 페이지 링크 (있는 경우)

#### 히어로 이미지
- 부드러운 일러스트 또는 앱 스크린샷
- 따뜻한 색상 팔레트
- 감정적 안전감을 주는 시각적 요소

### 2. 문제 정의 섹션

#### 섹션 제목
```
기존 일기 앱의 한계
```

#### 문제점 리스트
1. **부정적 감정 직면의 부담**
   - "쓰면 더 우울해진다"
   - 감사 찾기가 위선처럼 느껴짐

2. **완벽주의 압박**
   - "제대로 써야 한다는 부담"
   - 매일 써야 한다는 압박감

3. **즉각적 보상 부족**
   - "쓰고 나서 뭐가 달라지나?"
   - 기록만 하고 끝나는 공허함

4. **외로움**
   - "혼자만의 기록이라 공허하다"
   - 공감받지 못하는 느낌

5. **중단 후 죄책감**
   - "하루 빠지면 포기하게 된다"
   - 연속 기록에 대한 압박

#### 차별점 강조
```
우리는 다릅니다.
감사를 강요하지 않고, 
있는 그대로의 감정을 받아들이며,
당신이 여기 온 것만으로도 충분하다고 말합니다.
```

### 3. 핵심 기능 섹션

#### 섹션 제목
```
당신을 이해하는 기능들
```

#### 기능 1: 공감하는 AI 동반자
**제목**: "진정한 위로를 전달하는 AI"
**설명**: 
```
Claude AI 기반의 공감형 대화 인터페이스로,
당신의 감정을 깊이 이해하고 진정한 위로를 제공합니다.
감사를 찾지 못해도 괜찮다고 말해주는,
힘들 때 찾고 싶은 감정 동반자.
```

**주요 특징**:
- 감정을 인정하고 공감
- 작은 칭찬과 격려
- 감사를 강요하지 않음
- 생애주기별 맞춤 응답

**시각 자료**: AI 대화 예시 스크린샷

---

#### 기능 2: 부담 없는 입력
**제목**: "30초 안에 끝나는 간편 입력"
**설명**:
```
한 줄만 써도 충분합니다.
빠른 체크인부터 긴 글까지,
당신의 상황에 맞는 입력 방식을 선택하세요.
```

**주요 특징**:
- 빠른 체크인 (10초)
- 간단 기록 (30초)
- 자유 쓰기 (제한 없음)
- 음성 입력 지원 (선택적)

**시각 자료**: 입력 방식 비교 일러스트

---

#### 기능 3: 생애주기별 맞춤 경험
**제목**: "당신의 상황을 이해하는 앱"
**설명**:
```
사회 초년생, 취준생, 워킹맘, 빈둥지 부모...
당신의 생애주기에 맞는 질문과 응답을 제공합니다.
```

**주요 특징**:
- 온보딩에서 상황 파악
- 맥락 기반 질문
- 상황별 맞춤 응답
- 프로필 업데이트 가능

**시각 자료**: 생애주기별 예시 카드

---

#### 기능 4: 칭찬 시스템
**제목**: "시도한 것 자체를 칭찬"
**설명**:
```
앱을 연 것, 한 줄 쓴 것, 감정을 선택한 것...
모든 시도를 칭찬하고 인정합니다.
```

**주요 특징**:
- 앱 오픈 칭찬
- 입력 시도 칭찬
- 감정 인정 칭찬
- 며칠 못 써도 환영

**시각 자료**: 칭찬 카드 예시

---

#### 기능 5: 유연한 감사 찾기
**제목**: "강요하지 않는 감사 발견"
**설명**:
```
감사를 찾고 싶을 때만 찾으세요.
없어도 괜찮다고 말해주는,
유연한 감사 찾기 시스템.
```

**주요 특징**:
- 선택적 감사 찾기
- 작은 것도 감사로 인정
- 과거 기록 재발견
- 감사 패턴 분석

**시각 자료**: 감사 찾기 플로우

### 4. 사용자 여정 섹션

#### 섹션 제목
```
당신의 하루, 이렇게 기록하세요
```

#### 여정 단계

**1단계: 앱 열기**
```
"반가워, [이름]. 오늘 여기까지 오느라 수고했어"
```
- 따뜻한 환영 메시지
- 빠른 체크인 옵션

**2단계: 감정 토해내기**
```
사용자: "오늘도 실패했다"
AI: "정말 힘들었겠다. 그래도 포기하지 않은 너, 대단해"
```
- 부정적 감정도 받아들임
- 진정한 공감과 위로

**3단계: 선택적 감사 찾기**
```
AI: "혹시 오늘 하루 중에 괜찮았던 순간이 있었어?"
사용자: "없어"
AI: "괜찮아. 오늘은 네가 여기 온 것만으로도 충분해"
```
- 부드러운 질문
- 강요하지 않음

**4단계: 칭찬과 격려**
```
"오늘도 수고했어. 내일 또 보자"
```
- 작은 칭찬
- 다음을 위한 격려

#### 시각 자료
- 인터랙티브 플로우차트
- 실제 대화 예시
- 감정 변화 그래프
